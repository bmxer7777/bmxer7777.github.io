## About

This tracker follows the EcoCAR team's LYRIQ as it travels cross-country for California Air Resources Board testing. Built with Leaflet maps and OSRM routing.

## EcoCAR EV Challenge

The EcoCAR EV Challenge is a multi-year collegiate automotive engineering competition where students redesign and reengineer a Cadillac LYRIQ to improve efficiency and demonstrate advanced propulsion technologies.
